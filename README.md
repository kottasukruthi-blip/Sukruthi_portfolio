# Personal Portfolio Website

A modern, responsive, and interactive personal portfolio website built using **HTML, CSS, and JavaScript**. This portfolio showcases my education, technical skills, projects, certifications, achievements, and contact information in a clean and professional design.

## Features

* Responsive design for desktop, tablet, and mobile devices
* Interactive and modern user interface
* Hero section with introduction
* About Me section
* Skills with progress indicators
* Education timeline
* Projects showcase
* Certifications and achievements
* Contact section with social media links
* Smooth scrolling and animations
* Clean, organized, and maintainable code

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## Project Structure

portfolio/

## Purpose

The primary goal of this project is to present my professional profile, highlight my technical expertise, and showcase my work in an engaging and visually appealing manner. It also serves as a central platform for recruiters and employers to learn more about my skills and projects.
